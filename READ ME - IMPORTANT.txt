GenAudius version 2.5.0

If you're an existing customer and will be updating to GenAudius v2.5.0 from an older version, THIS IS A BREAKING CHANGE, PLEASE READ THE GUIDE BELOW.

In v2.5.0, the Authentication system has been migrated to a different library, so once you update the app ALL OF THE USERS' EXISTING PASSWORDS WILL STOP WORKING. You will need to do the following:
1. Backup your existing database in case anything goes wrong you can restore it.
2. In your environment variables file, add these 2 new env vars:
    - BETTER_AUTH_SECRET="your-better-auth-secret-key" (you can generate a secret from this page: https://better-auth.com/docs/installation#set-environment-variables)
    - BETTER_AUTH_URL="https://your-domain.com"
3. Migrate the db schema by running 'npm run db:push' in the root of the project directory

At this point, you will not be able to login to the app with your old password any longer, so you will have to instruct your users to reset their password via the https://your-domain.com/reset-password page.
However, please note for this to work you need to have the 'Mailing' already integrated within your app's admin dashboard settings.
